# Contributing

Thanks for contributing to `sudiptpa/paypal-notifications`.

## Development Setup

1. Install dependencies:

```bash
composer install
```

2. Run quality checks:

```bash
composer test
composer stan
```

## Contribution Rules

- Keep the package framework-agnostic.
- Add tests for any behavior changes.
- Update `README.md`, `CHANGELOG.md`, and `UPGRADE.md` when public behavior changes.
- Follow existing naming and API style aligned with PayPal webhook terminology.

## Pull Requests

- Make focused PRs with clear scope.
- Include a short summary and testing notes.
- Ensure CI is green before requesting merge.
